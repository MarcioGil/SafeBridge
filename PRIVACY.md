# Política de Privacidade

O SafeBridge respeita e protege a privacidade dos usuários. Todos os dados coletados são utilizados exclusivamente para fins de registro e acompanhamento de ocorrências, nunca sendo compartilhados com terceiros sem consentimento.

- Dados sensíveis são criptografados e armazenados de forma segura.
- O usuário pode optar por registrar denúncias de forma anônima.
- Os dados podem ser removidos mediante solicitação.
- Logs e backups não armazenam informações pessoais identificáveis.
- Consentimento explícito é solicitado para qualquer uso de dados além do registro da ocorrência.

Para dúvidas ou solicitações, entre em contato com o responsável pelo projeto.
