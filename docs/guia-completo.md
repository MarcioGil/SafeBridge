## Sobre o Projeto
SafeBridge é uma plataforma digital pensada para proteger, incluir e empoderar minorias vulneráveis — pessoas LGBTQIA+, pretas, PCDs, mulheres, indígenas e todos que enfrentam situações de risco ou exclusão. O app oferece um canal seguro, acessível e acolhedor para registro de ocorrências, denúncia de violência e busca de apoio, promovendo autonomia, privacidade e respeito.

## Como Usar
- Acesse o site: [SafeBridge](https://safebridge-clean-oy6v9r5r6-marciogils-projects.vercel.app)
- Instale como PWA: Abra no celular, clique em "Adicionar à tela inicial"
- Registre ocorrências: Preencha o formulário, envie anexos, escolha anonimato

## Acessibilidade
- Navegação por teclado e leitores de tela
- Contraste alto e foco visível
- Imagens e botões com descrição
- Layout responsivo para todos os dispositivos

## Segurança
- Dados protegidos e criptografados
- Autenticação segura (NextAuth.js)
- Backup automático do banco de dados
- Proteção contra ataques comuns (XSS, CSRF, SQL Injection)

## Arquitetura
- Next.js (frontend/backend)
- Prisma (ORM)
- Vercel (CI/CD)
- NextAuth.js (autenticação)
- Tailwind CSS (estilização)
- PWA (app mobile)
