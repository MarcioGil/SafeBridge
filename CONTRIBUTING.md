# Guia de Contribuição

Obrigado por querer contribuir com o SafeBridge!

## Como contribuir
- Faça um fork do repositório
- Crie uma branch descritiva para sua contribuição
- Siga o padrão de código, lint e testes
- Abra um Pull Request detalhando sua proposta
- Aguarde revisão e feedback

## Regras para colaboradores
- Respeite o [Código de Conduta](CODE_OF_CONDUCT.md)
- Não exponha dados sensíveis ou segredos
- Priorize acessibilidade, segurança e privacidade
- Testes e documentação são obrigatórios para novas features
- Reporte bugs e vulnerabilidades de forma responsável

## Dúvidas?
Abra uma issue ou entre em contato pelo LinkedIn do autor.
